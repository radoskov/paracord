\section{API and User Interface}

The API is versioned under \texttt{/api/v1}. The frontend should provide login, library table, shelf/rack view, file view, PDF reader, citation graph, citation context panel, duplicate review queue, export dialog, import status, audit log, and AI/topic views.

\subsection{Search scopes}

Many features should operate over common scopes: full library, rack, shelf, search result, selected works, and import batch.

\subsection{Citation graph controls}

The graph should allow toggling local-only vs external references, full library vs rack vs shelf, citation context display, duplicate-version collapse, color by tag, and grouping by shelf.
