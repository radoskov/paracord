\section{Glossary}

\begin{description}[style=nextline]
  \item[Work] Conceptual scholarly paper.
  \item[Version] Specific edition or publication state of a work.
  \item[File] Physical file instance.
  \item[File segment] Page range within a file that maps to one work.
  \item[Shelf] Collection of works.
  \item[Rack] Collection of shelves.
  \item[Teleport] Copying a file from local agent or server root into managed server storage.
  \item[Citation context] Text and PDF coordinates around an in-text citation mention.
\end{description}
