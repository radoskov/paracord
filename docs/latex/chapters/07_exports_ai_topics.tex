\section{Exports, AI Summaries, and Topics}

\subsection{Citation exports}

Required export scopes are individual work, selected works, shelf, rack, search result, citation graph neighborhood, and missing references. Required formats include BibTeX, BibLaTeX, RIS, CSL JSON, Markdown, HTML, and plain-text bibliography.

\subsection{Summaries}

Summaries must be stored separately by type: extracted abstract, local AI summary, external AI summary, human summary, and imported summary. AI summaries must record model name, prompt version, date, and source sections.

\subsection{Topics}

Topic modeling can be run over the full library, a rack, a shelf, a search result, or an import batch. Topic suggestions should not automatically overwrite user tags.
