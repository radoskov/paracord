\section{Implementation Work Split}

Recommended coding-agent areas:

\begin{enumerate}[noitemsep]
  \item Backend platform and database.
  \item Authentication, authorization, and audit logging.
  \item Local workstation agent.
  \item Ingestion, storage, and duplicate/version detection.
  \item GROBID extraction and metadata enrichment.
  \item Citation graph and citation summaries.
  \item Frontend UI.
  \item Citation export.
  \item Local AI, embeddings, and topic modeling.
  \item Documentation, tests, and developer experience.
\end{enumerate}

Agents should update \texttt{PROGRESS.md} and add handoff notes after completing tasks.
